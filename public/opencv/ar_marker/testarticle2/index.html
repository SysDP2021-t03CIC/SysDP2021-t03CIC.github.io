ARマーカーの認識

ARマーカーの認識から追従対象の距離・角度の推定についての解説

  • 座標
    検出できた各ARマーカーからx座標,y座標,z座標を得ることができる。
    その各座標値の平均を追従対象の座標としてます。
  • 回転
    おおよそ上記の座標と同じ。
    ただし、計算時に 回転の値 + 90 × ID として平均を求める。
    IDが0のARマーカーに関しては回転の値が負の時、+360する。
    x軸回転,y軸回転,z軸回転が得られるが、利用するのはy軸回転のみ。
  • 方向
    得られた座標の値から
    左右の方向 : arctan(x / z)
    上下の方向 : arctan(y / Z)
    で求められる。
  • 距離
    座標の値から求められる。
